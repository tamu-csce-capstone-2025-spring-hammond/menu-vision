Class

DishMapping

A class responsible for managing dish model data and state tracking in the application.

Declaration

class DishMapping

Overview

DishMapping provides a centralized store for dish model data, organized by dish name, and includes state flags for tracking download and loading processes. It also maintains a counter for model download progress.
