CheckboxField

A customizable checkbox field with accompanying text

Declaration

@MainActor struct CheckboxField

Overview

This view provides a checkbox control with associated text label. The text can include specially formatted sections like “Terms and Conditions” and “Privacy Policy” which will be displayed in bold.

Example usage:

@State private var isChecked = false

CheckboxField(
    isChecked: $isChecked,
    text: "I've read and agree with the Terms and Conditions and the Privacy Policy."
)
