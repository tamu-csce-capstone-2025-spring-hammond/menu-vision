HomeView

The main view for the MenuVision application, providing a tabbed interface for navigation.

Declaration

@MainActor struct HomeView

Overview

This view manages the app’s primary navigation using a TabView, allowing users to switch between different sections of the application.

The view uses a custom tab bar appearance that adapts to light and dark modes.
