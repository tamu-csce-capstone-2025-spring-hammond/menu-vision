SignUpData

A class for managing sign-up form data

Declaration

class SignUpData

Overview

This observable object provides properties for storing and managing all the data collected during the user signup process, including dietary restrictions, cuisine preferences, personal information, and authentication credentials.

Example usage:

let signUpData = SignUpData()
signUpData.first_name = "John"
signUpData.email = "john@example.com"
