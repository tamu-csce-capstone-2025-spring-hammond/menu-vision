Instance Method

request(endpoint:method:body:headers:useBackup:completion:)

Performs a network request with configurable parameters and fallback mechanism.

Declaration

func request(endpoint: String, method: String = "GET", body: Data? = nil, headers: [String : String] = [:], useBackup: Bool = false, completion: @escaping (Result<Data, Error>) -> Void)

Parameters

endpoint

The specific API endpoint to be called

method

The HTTP method (default is “GET”)

body

Optional request body data

headers

Optional HTTP headers

useBackup

Flag to determine whether to use backup URL (default is false)

completion

Closure to handle the result of the network request

Discussion

Note

Automatically handles retrying with backup URL if the primary request fails
