Class

LoginHandler

A singleton class that handles user authentication operations for the MenuVision app.

Declaration

class LoginHandler

Overview

LoginHandler provides methods for validating user credentials, fetching AWS credentials after successful login, and retrieving stored AWS credentials from UserDefaults.
