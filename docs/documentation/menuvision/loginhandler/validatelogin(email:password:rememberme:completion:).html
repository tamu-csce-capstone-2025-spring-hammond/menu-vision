Instance Method

validateLogin(email:password:rememberMe:completion:)

Validates user login credentials against the backend API.

Declaration

func validateLogin(email: String, password: String, rememberMe: Bool, completion: @escaping (Bool, String?, Int?) -> Void)

Parameters

email

The user’s email address.

password

The user’s password.

rememberMe

Boolean indicating if the user’s login should be persisted.

completion

A closure that receives login success status, optional error message, and optional user ID. Called on the main thread when authentication completes.

Discussion

This method sends the user’s email and password to the authentication endpoint, processes the response, and handles storing user ID in UserDefaults if “remember me” is enabled.
