Class

ModelFileManager

A singleton class that manages 3D model files, including downloading, storage, and cleanup.

Declaration

class ModelFileManager

Overview

This class handles the transfer of 3D model files between AWS S3 storage and the local device filesystem. It provides methods to download all models for a restaurant and clear local storage when needed.
