Instance Method

clearAndDownloadFiles(for:dishMapping:)

Clears all local files and downloads model files for a specific restaurant.

Declaration

func clearAndDownloadFiles(for restaurantID: String, dishMapping: DishMapping) async -> [DishData]

Parameters

restaurantID

The unique identifier of the restaurant.

dishMapping

The DishMapping instance to update with download progress.

Return Value

An array of DishData objects representing the downloaded models.

Discussion

This method removes all previously downloaded files from the documents directory and then downloads all models associated with the specified restaurant from S3.
