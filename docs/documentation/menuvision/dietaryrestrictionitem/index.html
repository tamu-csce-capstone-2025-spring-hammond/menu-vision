DietaryRestrictionItem

A selectable item for dietary restriction options

Declaration

@MainActor struct DietaryRestrictionItem

Overview

This view displays a dietary restriction option as a selectable button with visual feedback for the selected state. When selected, the item is highlighted with an orange background.

Example usage:

DietaryRestrictionItem(
    title: "Dairy Allergy",
    isSelected: true,
    onTap: {
        // Handle selection
    }
)
